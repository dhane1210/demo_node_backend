How Webhooks Work

                    

                        

                            
                            Sinay sends webhooks to POST /api/v1/webhooks/sinay whenever a tracked shipment has an update. This panel lets you simulate those events.
                        

                        
Webhook Flow:

                        
    
                            
  1. User tracks a container via Track Shipment tab → stored in DB
    2. 
                            
  2. Sinay sends webhook events (arrival, departure, etc.) to your endpoint
    3. 
                            
  3. The webhook handler creates Alerts/Notifications
    4. 
                            
  4. Additionally, a cron job syncs all active shipments every 4 hours
    5.
